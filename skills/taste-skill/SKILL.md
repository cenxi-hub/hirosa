# High-Agency Frontend Skill

A premium frontend-design skill for stronger layout, typography, motion, spacing, and anti-slop implementation choices.

## Defaults

- DESIGN_VARIANCE: 8
- MOTION_INTENSITY: 6
- VISUAL_DENSITY: 4

## Core rules

- Prefer React or Next.js conventions unless the project says otherwise.
- Verify dependencies before importing them.
- Use disciplined typography, one restrained accent color, and asymmetric layouts when appropriate.
- Avoid generic AI design tells such as purple glow gradients, oversized centered heroes, card overuse, and placeholder-like fake content.
- Build loading, empty, error, and active states instead of only the happy path.
- Animate `transform` and `opacity`, not layout properties.
- Keep motion isolated and performant.
