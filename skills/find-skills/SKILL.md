# Find Skills

This skill helps discover and install skills from the open agent skills ecosystem.

## Core guidance

1. Understand the user's need.
2. Check popular sources first.
3. Search when needed.
4. Verify quality before recommending: installs, source reputation, and GitHub stars.
5. Present options clearly and offer to install.
